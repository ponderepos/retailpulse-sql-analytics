# RetailPulse — SQL Data Analytics Project

This project simulates a real-world e-commerce analytics database for a mid-size Indian retail company.

The dataset contains customer, product, supplier, and order information used for business analysis.

---

## Database Schema

Tables used in this project:

- customers
- suppliers
- products
- orders
- order_items
- returns

---

## Skills Demonstrated

SQL skills used:

- Joins
- Aggregations
- Window Functions
- CTEs
- Data Quality Checks
- Business Analytics Queries

---

## Business Questions Answered

Examples:

- Top revenue generating cities
- Product profitability analysis
- Customer spending segmentation
- Month-over-month revenue growth
- Return rate analysis
